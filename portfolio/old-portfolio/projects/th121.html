Videogames are important parts of any North-American highschool teenager's highschool life. I find enjoyment not only in playing, but creating them as well. During the summer of grade 10 two friends and I formed a programming team, and decided to create a videogame in honor of our favourite Japanese videogame franchise, Project Touhou. This project was meant to be a tangent off of the original franchise. Our team now refers ourselves as LANEX-DOUJIN TEAM. "Doujin" meaning "fan made work".

Key Features:

  • Cross-platform
  • Coded in Java (WHO SAID YOU CAN'T MAKE GAMES IN JAVA!?)
  • Uses LWJGL's Java-OpenGL binding libraries for high graphical performance
  • IT'S A GAME! :D

Design Challenges

As a student who does not hesitate to call himself a programmer and a designer, this project has presented me with a number of difficulties and design challenges. The project was created with the following high-level objectives in mind:

  • Must be efficient.
  • Must be well organized.
  • Should be easy to implement new features
  • Should provide a platform for future development
  • Should look PRETTY!
